# Lazer Assembler 700

### Подготовил студент ИКБО-23-24 Федоров Т. В.

## Этап 3

Интерпретатор и операции с памятью

CLI способен читать двоичный файл с командами и исполнять перемещения в памяти, записывая срез памяти в csv файл

<hr>

## Структура проекта

- Класс `ArgParse` — обработчик аргументов
- Класс `LazerSystem` — исполнитель команд
- Изолированный класс `Command` — абстрактное представление всех комманд

<hr>

## Принимаемые аргументы

- Файл-источник ввода (txt/bin)
- Файл с результатом работы программы (bin/csv)
- Режим тестирования (-t, --test)
- Диапазон адресов памяти в формате XX:XX (-r, --range)

<hr>

## Команды

- **LOAD** - Загрузка константы (3 аргумента)
- **READ** - Чтение значения из памяти (3 аргумента)
- **WRITE** - Запись значения в память (4 аргумента)
- **NEGATIVE** - Унарный минус (3 аргумента) 
- **HISTORY** - История введённых значений (без аргументов)
- **ASSEMBLED** - Количество ассемблированных команд (без аргументов)
- **BYTES** - Вывод всех ассемблированных команд (без аргументов)
- **EXIT** - Завершение работы (без аргументов)
